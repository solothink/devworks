# **App Name**: DevWorks Portfolio

## Core Features:

- Hero Section: A visually appealing hero section with a professional headline and clear call-to-action buttons (Hire Me/Get a Quote).
- Services Showcase: Dedicated pages for each service (Web Development, Mobile App Development, Custom Software & APIs, AI & Blockchain Solutions, Maintenance & Support) with detailed descriptions and example packages.
- Portfolio Display: Interactive portfolio showcasing past projects with images, descriptions, and links (GitHub/Live links). Case studies for selected projects to demonstrate expertise.
- Project Uploader: A secure form allowing potential clients to upload project requirements (details, PDFs, images, docs).
- Client Success Story Selector: A tool to pick a suitable client success story to add to a testimonial. An LLM tool chooses one of several user stories provided.
- Contact Gateway: A contact form with essential fields (Name, Email, Message) along with direct contact information (Email & Phone number) and social media links (LinkedIn, GitHub, Twitter).
- Responsive Layout: Fully responsive design adapting seamlessly to various devices (desktop, tablet, mobile).

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey professionalism and innovation.
- Background color: Very light gray (#F0F0F0) for a clean and modern look.
- Accent color: Soft violet (#8A2BE2) to highlight key elements and call-to-actions.
- Headline font: 'Space Grotesk', a sans-serif font that will impart a techy, scientific look.
- Body font: 'Inter', a sans-serif font, suitable for body text
- Employ a clean, minimal grid layout with generous whitespace for enhanced readability and visual appeal.
- Incorporate subtle animations for smooth transitions and interactive elements to enhance user engagement.
